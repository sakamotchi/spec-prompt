# テスト手順書 - Phase 3-F: 多言語対応（日本語・英語）

## 概要

多言語対応（日本語・英語切り替え）の手動テスト手順を記載します。

## 前提条件

- `npx tauri dev` でアプリが起動していること
- プロジェクトフォルダが開かれていること

---

## 手動テスト

### ケース 1: デフォルト言語が日本語であること

**手順:**

1. アプリを初回起動する（またはlocalStorageをクリアして再起動）

**期待結果:**

- UIのすべてのラベルが日本語で表示される
- ツリーパネルの「フォルダを開いてください」が表示される

**確認結果:**

- [ ] OK / NG

---

### ケース 2: 設定モーダルから英語に切り替える

**手順:**

1. 設定アイコンをクリックして設定モーダルを開く
2. 「言語」セクションで「English」を選択する

**期待結果:**

- モーダルを閉じずとも即時に全UIが英語に切り替わる
- 「外観設定」→「Appearance」、「フォルダを開いてください」→「Open a folder to get started」に変わる

**確認結果:**

- [ ] OK / NG

---

### ケース 3: 英語表示でコンテキストメニューを確認する

**手順:**

1. 英語に切り替えた状態でファイルを右クリックする

**期待結果:**

- メニュー項目が英語で表示される（New File / New Folder / Rename / Delete 等）

**確認結果:**

- [ ] OK / NG

---

### ケース 4: 英語表示でショートカット一覧モーダルを確認する

**手順:**

1. 英語に切り替えた状態で `?` キーを押す

**期待結果:**

- モーダルタイトルが「Keyboard Shortcuts」と表示される
- カテゴリ・ショートカットラベルがすべて英語

**確認結果:**

- [ ] OK / NG

---

### ケース 5: 英語表示で削除ダイアログを確認する

**手順:**

1. 英語に切り替えた状態でファイルを右クリック →「Delete」を選択する

**期待結果:**

- ダイアログタイトルが「Delete File」、ボタンが「Cancel」/「Delete」と表示される

**確認結果:**

- [ ] OK / NG

---

### ケース 6: 英語表示で PathPalette を確認する

**手順:**

1. 英語に切り替えた状態で `Ctrl+P` を押す

**期待結果:**

- プレースホルダーが「Search files to insert into terminal...」と表示される
- 「Esc で閉じる」→「Esc to close」に変わる

**確認結果:**

- [ ] OK / NG

---

### ケース 7: 英語から日本語に戻す

**手順:**

1. 設定モーダルを開き「日本語」を選択する

**期待結果:**

- 即時に全UIが日本語に戻る

**確認結果:**

- [ ] OK / NG

---

### ケース 8: 言語設定が再起動後も維持される

**手順:**

1. 英語に切り替える
2. アプリを再起動する

**期待結果:**

- 再起動後も英語表示が維持される

**確認結果:**

- [ ] OK / NG

---

### ケース 9: 動的文字列（件数）の補間が正しく動作する

**手順:**

1. 英語表示でツリーパネルから複数ファイルを選択（Shift+クリック）
2. 右クリックメニューを開く

**期待結果:**

- 「Insert All 3 Selected」のように件数が正しく補間される

**確認結果:**

- [ ] OK / NG

---

### ケース 10: DocStatus ラベルの翻訳確認

**手順:**

1. 英語に切り替えた状態でステータス付き `.md` ファイルを右クリック →「Status」サブメニューを開く

**期待結果:**

- サブメニューに「Draft」/「Reviewing」/「Approved」が表示される

**確認結果:**

- [ ] OK / NG

---

## エッジケース

| ケース | 期待動作 | 確認結果 |
|--------|---------|---------|
| MDプレビューの内容（日本語文書） | 翻訳されず、ファイルの内容がそのまま表示される | |
| ターミナルのシェル出力 | 翻訳されない | |
| コードビューアのシンタックスハイライト | 影響なし | |
| 翻訳キーが存在しない場合（フォールバック） | フォールバック言語（日本語）のテキストが表示される | |

- [ ] ファイルツリーが正常に表示される
- [ ] コンテンツビューアが正常に動作する
- [ ] ターミナルが正常に起動・入力できる
- [ ] キーボードショートカットが動作する
- [ ] テーマ切り替えが動作する
- [ ] 拡張子別アイコンが引き続き表示される（Phase 3-D）

# テスト手順書 - alacritty-terminal Phase 3（スクロールバック・選択コピー・パス挿入）

## 概要

Phase 3 の手動テストは Canvas レンダラーをデフォルトで使用する（Feature Flag 廃止後は通常起動）。
Feature Flag 廃止前は `VITE_USE_CANVAS_RENDERER=true npx tauri dev` で起動する。

## 前提条件

- `npx tauri dev` でアプリが起動していること（Phase 3 完了後は Feature Flag 不要）
- Phase 3 完了前: `VITE_USE_CANVAS_RENDERER=true npx tauri dev`

---

## 手動テスト - スクロールバック（3-A）

### ケース 1: マウスホイールで過去の出力を遡れる

**手順:**

1. ターミナルを開く
2. `yes | head -200` を実行して大量のテキストを出力する
3. マウスホイール（またはトラックパッド上スワイプ）でスクロールする
4. 下にスクロールして末尾に戻る

**期待結果:**

- 過去の出力が Canvas 上に表示される
- スクロールバーが右端に表示され、位置が変化する
- 末尾に戻ると最新の出力が表示される

**確認結果:**

- [x] OK

---

### ケース 2: キー入力でスクロール位置が末尾に戻る

**手順:**

1. `yes | head -200` で大量出力する
2. ホイールで先頭付近にスクロールする
3. 任意のキー（例: `e`）を押す

**期待結果:**

- キーを押した瞬間にスクロール位置が末尾に戻る
- 入力した文字がプロンプトに表示される

**確認結果:**

- [x] OK

---

### ケース 3: scrollback が 10,000 行蓄積される

**手順:**

1. `seq 1 10100` を実行する（10,100 行出力）
2. ホイールで先頭方向に大きくスクロールする

**期待結果:**

- 最大 10,000 行の履歴が保持されており、先頭付近まで遡れる
- 10,000 行を超えた分は古い行から消える

**確認結果:**

- [x] OK

---

## 手動テスト - テキスト選択・コピー・ペースト（3-B）

### ケース 4: マウスドラッグでテキストを選択できる

**手順:**

1. `echo "Hello World"` を実行する
2. 出力テキストの先頭でマウスボタンを押し、末尾までドラッグして離す

**期待結果:**

- ドラッグ中に選択範囲が紫系半透明ハイライトで表示される
- マウスを離しても選択状態が維持される

**確認結果:**

- [x] OK

---

### ケース 5: Cmd+C で選択テキストをコピーできる

**手順:**

1. `echo "copy test"` を実行する
2. `copy test` の部分をドラッグで選択する
3. Cmd+C を押す
4. テキストエディタなど別アプリで Cmd+V を押す

**期待結果:**

- クリップボードに `copy test` が入っている
- テキストエディタに `copy test` が貼り付けられる

**確認結果:**

- [x] OK

---

### ケース 6: Cmd+V でクリップボード内容をペーストできる

**手順:**

1. システムのクリップボードに任意のテキスト（例: `paste test`）をコピーしておく
2. ターミナルにフォーカスして Cmd+V を押す

**期待結果:**

- PTY に `paste test` が送信され、プロンプトに表示される

**確認結果:**

- [x] OK

---

### ケース 7: キー入力で選択が解除される

**手順:**

1. テキストをドラッグで選択する
2. 任意のキーを押す

**期待結果:**

- 選択ハイライトが消える

**確認結果:**

- [x] OK

## 手動テスト - パス挿入（3-C）

### ケース 8: Cmd+Click でパスがターミナルに挿入される

**手順:**

1. プロジェクトを開き、ファイルツリーを表示する
2. ターミナルタブを開く
3. ファイルツリーの任意のファイルを Cmd+Click する

**期待結果:**

- ターミナルのプロンプトにファイルパスが挿入される（末尾にスペースあり）

**確認結果:**

- [x] OK

---

### ケース 9: PathPalette（Ctrl+P）からパスを挿入できる

**手順:**

1. ターミナルタブにフォーカスした状態で Ctrl+P を押す
2. ファイル名の一部を入力して候補を絞り込む
3. Enter を押して選択する

**期待結果:**

- ターミナルのプロンプトに選択したファイルのパスが挿入される

**確認結果:**

- [x] OK

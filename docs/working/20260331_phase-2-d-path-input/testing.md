# テスト手順書 - Phase 2-D: パス入力支援

## 概要

Ctrl+クリック・右クリックメニュー・PathPalette・複数選択によるパス挿入機能のテスト手順を記載します。

## 前提条件

- `npx tauri dev` でアプリが起動していること
- テスト用プロジェクトとして `spec-prompt` リポジトリを開いておく
- ターミナルタブが1つ以上開いており、シェルが起動していること

---

## 手動テスト

### ケース 1: Ctrl+クリックでファイルパスを挿入

**手順:**

1. ターミナルを表示した状態にする（コンテンツモードでも可）
2. ファイルツリーから任意のファイル（例: `README.md`）を **Ctrl+クリック**（macOS: Cmd+クリック）する

**期待結果:**

- ターミナルのプロンプトにファイルパスが挿入される（例: `README.md `）
- コンテンツビューアでファイルが開かれない（通常クリックの動作が起きない）

**確認結果:**

- [ ] OK / NG

---

### ケース 2: Ctrl+クリックでフォルダパスを挿入

**手順:**

1. ファイルツリーから任意のフォルダ（例: `src`）を **Ctrl+クリック** する

**期待結果:**

- ターミナルにフォルダパスが挿入される（例: `src `）

**確認結果:**

- [ ] OK / NG

---

### ケース 3: 通常クリックが妨げられないこと

**手順:**

1. ファイルツリーから `.md` ファイルを **通常クリック** する

**期待結果:**

- コンテンツビューアでファイルが開く（Ctrl+クリックとの干渉なし）

**確認結果:**

- [ ] OK / NG

---

### ケース 4: 右クリックメニューからパスを挿入

**手順:**

1. ファイルツリーのファイルを **右クリック** する
2. 表示されたメニューから「パスをターミナルに挿入」をクリックする

**期待結果:**

- コンテキストメニューが表示される
- 「パスをターミナルに挿入」をクリックするとパスがターミナルに挿入される
- メニューが自動的に閉じる

**確認結果:**

- [ ] OK / NG

---

### ケース 5: コンテキストメニューを Esc で閉じる

**手順:**

1. ファイルを右クリックしてメニューを表示する
2. **Esc** キーを押す

**期待結果:**

- メニューが閉じる
- パスは挿入されない

**確認結果:**

- [ ] OK / NG

---

### ケース 6: コンテキストメニューをクリック外で閉じる

**手順:**

1. ファイルを右クリックしてメニューを表示する
2. メニュー外の領域をクリックする

**期待結果:**

- メニューが閉じる
- パスは挿入されない

**確認結果:**

- [ ] OK / NG

### ケース 7: Ctrl+P でパス検索パレットが開く

**手順:**

1. **Ctrl+P**（macOS: Cmd+P）を押す

**期待結果:**

- 画面中央にパス検索パレットがオーバーレイ表示される
- 入力フィールドに自動的にフォーカスが当たる

**確認結果:**

- [ ] OK / NG

---

### ケース 8: PathPalette でファイルを検索して挿入

**手順:**

1. Ctrl+P でパレットを開く
2. 「README」と入力する
3. 候補一覧から `README.md` を **Enter** または **クリック** で選択する

**期待結果:**

- 入力に応じてリアルタイムに候補が絞り込まれる
- 選択するとパスがターミナルに挿入される
- パレットが自動的に閉じる

**確認結果:**

- [ ] OK / NG

---

### ケース 9: PathPalette を Esc で閉じる

**手順:**

1. Ctrl+P でパレットを開く

**確認結果:**

- [ ] OK / NG

### ケース 10: PathPalette で検索結果がない場合

**手順:**

1. Ctrl+P でパレットを開く
2. 存在しないファイル名（例: `xyzxyzxyz`）を入力する

**期待結果:**

- 「一致なし」または候補ゼロの状態が表示される

**確認結果:**

- [ ] OK / NG

---

### ケース 11: 複数ファイル選択・一括挿入

**手順:**

1. ファイルツリーでファイル A をクリックする
2. **Ctrl+クリック**（macOS: Cmd+クリック）でファイル B を追加選択する
3. 選択中のいずれかのファイルを右クリックし、「選択中のパスをすべて挿入」をクリックする

**期待結果:**

- ファイル A・B のパスがスペース区切りでターミナルに挿入される（例: `fileA fileB `）

**確認結果:**

- [ ] OK / NG

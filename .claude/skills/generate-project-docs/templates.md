# テンプレート集

`docs/projects/{YYYYMMDD}-{要件名}/` に生成する 3 ドキュメントのテンプレート。

**重要**: これらは雛形です。実際の生成時は、ヒアリング内容と Explore 調査結果を踏まえて中身を書き起こしてください。機密情報（APIキー、トークン、個人情報、社内固有URL等）は記載禁止。

---
